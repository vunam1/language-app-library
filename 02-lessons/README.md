# Lessons Directory Guide

## 1. Tiêu đề
Lessons Directory Guide

## 2. Mục đích của thư mục 02-lessons
Thư mục 02-lessons chứa toàn bộ lesson chi tiết của app. Đây là nơi chuyển curriculum thành bài học thật.

## 3. Vai trò của tầng lessons
Tầng lessons biến curriculum thành lesson cụ thể, bám lesson schema. Nó là cầu nối giữa curriculum và trải nghiệm học thực tế.

## 4. Quy ước tổ chức thư mục theo từng tháng
Mỗi tháng có thư mục riêng như 02-lessons/month-01/. Mỗi thư mục tháng chứa lesson file và README.md điều hướng.

## 5. Quy ước đặt tên lesson
Lesson dùng format lesson-01.md, lesson-02.md để thống nhất và dễ quản lý.

## 6. Lesson phải bám App Vision, Learning Principles, Curriculum tháng và Lesson Schema
Mọi lesson phải tuân thủ các nguyên tắc này để nhất quán.

## 7. Một lesson tốt cần có gì
Lesson tốt cần mục tiêu rõ, input, luyện tập có kiểm soát, output, review, test nhỏ, daily score signals.

## 8. Cách đọc thư mục này
Đọc README tháng trước để hiểu context, sau đó đọc lesson theo thứ tự.

## 9. Trạng thái hiện tại
Đã có month-01 với 12 lesson làm mẫu chuẩn.

## 10. Kết luận ngắn
Thư mục 02-lessons là trung tâm trải nghiệm học, giữ quy ước giúp dự án dễ mở rộng.
